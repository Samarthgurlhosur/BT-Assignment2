! START HERE
============

This file guides you through starting the Land Registration System immediately.

STEP 1: INSTALL DEPENDENCIES (1 minute)
========================================
Open PowerShell in the BT folder and run:

    pip install -r requirements.txt

This will install all 20+ required packages.


STEP 2: RUN THE APPLICATION (30 seconds)
========================================
In the same PowerShell, run:

    python app.py

You should see:
    WARNING: This is a development server. Do not use it in production.
    Running on http://127.0.0.1:5000

The app is now running!


STEP 3: OPEN IN BROWSER (10 seconds)
====================================
Go to: http://localhost:5000

You should see the login page.


STEP 4: CREATE TEST DATA (1 minute)
===================================
In a new PowerShell in the BT folder, run:

    flask --app app seed_data

This creates:
- 2 test users (owner and buyer)
- 3 sample lands
- All ready for testing
